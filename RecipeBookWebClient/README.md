# RecipeBook Web Client

Простой React + Vite клиент для управления продуктами и блюдами через API.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Тесты

```bash
npm run test
```

## Структура

- `src/App.tsx` — маршруты и основная навигация
- `src/pages` — страницы продуктов и блюд
- `src/api.ts` — HTTP-интерфейс к API
- `src/types.ts` — типы данных из OpenAPI
- `src/utils.ts` — подсчет КБЖУ и макросы
